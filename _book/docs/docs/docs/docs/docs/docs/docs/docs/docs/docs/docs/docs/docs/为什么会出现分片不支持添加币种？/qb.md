# 为什么会出现分片不支持添加币种？

区块链内不同的币种有不同的技术，新接入的链分片架构与原链可能会不兼容，因此需要重新计算分片才可以添加该币种。遇到此情况时只需要重置分片便可以，重置分片的操作步骤可参考[这里](https://support.Nexa.com/Nexa-wallet-guide/zh-cn/wallet_and_transaction/shard)。
